Mobeom Server
==========
 Project
---------

+ [Client](https://github.com/Mobeom/MoBeom) 
+ Server

Info
------
- Framework : **Node.js Express**
- Database : **MySQL**
 

Installation
--------
before getting start, you should install modules by using npm
```js
npm install
```
Getting Start
-------
souce code for starting the server is in /index.js 
```js
node app.js
````
or if you installed nodemon 
```js
npm run start
```
